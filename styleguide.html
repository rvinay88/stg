Logo Usage Guidelines

  1. No elements of the logo artwork may be recreated, deleted, cropped,or reconfigured.
  2. All logo artwork is provided as Adobe Illustrator,based EPS or pdf version
  3. Logo must be uniformly scaled. The text within the logo must be readable at all times. Scaling up does not pose any restrictions.
    However, scaling down requires that the text on the logo be readable with minimal effort
  4. Logo artwork should appear against a solid white background to ensure maximum and proper contrast.
  5. It can be placed over images but the background should not be too crowded and the logo should be clearly visible.
  6. Logo should be surrounded with a minimum amount of whitespace to differentiate between logo and other elements

Dont's for the logo

  1. Don't change the logo's orientation
  2. Don't bevel or emboss the logo
  3. Don't crop the logo in anyway
  4. Don't change the logo's colors
  5. Don't add glow effect to the logo
  6. Don't place logo in a crowded background Don't scale the logo disproportionately

Font Guidelines

  1. Fonts used are Helvetica and Futuralight with varying sizes
  2. Colors
    Base color for titles and primary information - Black #000
    Secondary color for Supplemental information - Grey #666
  3. Default font size - 12px
    Small font size - 11px
    Medium font size - 13px
    Large font size - 15px
  4. Links
    Standard links are black and no underline. All links underline on hover
  5. Captilization rules
    Designer Titles, Section Headers, Menu Links, Action Links (VIEW FULL SCREEN, WHAT SIZE AM I?), Button texts are all capitals
    Designer locations, Product Names are Sentence Case
  6. Black bars are used as section delimiters denoting a block of content as a group and separating it from other blocks on the page
